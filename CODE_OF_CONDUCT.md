# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, constructive, and focused on improving the project. Unacceptable
behavior includes harassment, discriminatory language, and personal attacks.

Project maintainers may remove comments, issues, pull requests, or other
contributions that do not align with these expectations.
